项目资料管理

功能说明​

“项目资料”主要用于设定管理本店系统中服务项目、服务项目的价格及做项目给积分等。

操作步骤​

1.添加服务
 • 在“MaSe”导航栏中打开“设置-基础设置-项目管理”功能，进入“项目列表”页面。
• 首先点击右上角⚙️-类别管理，关闭自己不需要的类别，新建自己需要的类别。
• 点击右上角“新增”按钮打开“添加新项目”页面。填写“项目名称“，”项目编号“，“项目分类“，”价格类型“等相关内容，带有用红色”*️⃣“的为必填，其他为选填，完成后点击“保存”完成设置。

2.编辑服务
 • 在“项目列表”页面，点击要修改的服务后的“编辑”按钮，修改里面的内容，完成后点击“保存”完成设置。

3.快速设置服务
 • 在“项目列表”页面，点击右上角的“快速设置”按钮，选择“设置范围“，选择“设置内容“，输入要设置的值，输入”确认“点击“确定”按钮，可以批量修改服务的内容。

4.导出服务
 • 在“项目列表”页面，点击右上角的“导出为”按钮，系统将自动导出所有服务为一个表格。
• 在系统主页右上角“⏬”按钮处，下载刚刚导出的报表

5.停用服务
 • 在“项目列表”页面，点击要修改的服务后的“状态”按钮，关闭按钮后，该项目在“收银”，“日程“等业务页面都不会再显示，但不影响已购买权益继续消耗。

6.关闭服务在线预约
 • 在“项目列表”页面，点击要修改的服务后的“线上预约”按钮，关闭按钮后，该项目在线预约时不会再出现，客人无法预约该项目。

7.修改服务顺序
 • 在“项目列表”页面，点击要修改的服务前的“排序”的数字，数字越小，排序越靠前，该顺序影响门店收银，和客人在线预约等的服务展示顺序。

温馨提示​

  1. 系统中已经使用的项目资料不允许删除；
  2. 项目名称支持设置多种语言显示，可以在“设置-门店管理-门店资料-启用第二语言”中开启；
  3. 项目类别需先在“项目列表-类别管理”中先建立好项目类别，再进入本作业进行选择。
