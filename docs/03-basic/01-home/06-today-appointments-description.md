# 今日预约说明

首页里的今日预约一栏，展示今日还没有开单的预约，顺序根据临近时间排序。<br/>

---

## 功能入口

进入路径：

**系统后台** → **首页** → **今日预约**<br/>
可点击相应预约后的“**开单**”按钮跳转去收银页面完成收款<br/>
