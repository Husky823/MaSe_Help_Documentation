# 员工排行说明

服首页上员工排行一栏的排序规则，是按照本月员工给门店带来的营业额，从高到低排序的。记录员工给门店带来的营业额，需要在收银界面选择对应的服务/销售员工。<br/>

---

## 功能入口

进入路径：

**系统后台** → **首页** → **员工排行**
