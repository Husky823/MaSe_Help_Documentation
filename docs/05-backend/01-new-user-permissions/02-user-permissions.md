# 用户权限管理
## 功能说明

“用户权限”主要功能是用户资料的管理，用户只需要设定该用户的**角色**即可以由程序根据角色权限设定自动获取**用户权限**；<br/>
如果有**多位人员**需要使用“MaSe”，您可以在用户权限中为每位人员设置一个**单独的登录账户**，还可以根据人员的岗位设置**不同的**使用权限。<br/>

**角色权限**分为下面5个部分：<br/>
“**可访问门店**”是指该角色可以登录使用的门店。<br/>
“**MaSe权限**”是指该角色可使用的系统菜单及功能权限。<br/>
“**管理端权限**“是指该角色可使用的”**MaSe Center**“手机APP功能权限。<br/>
“**支付方式**”是指该角色可见支付方式。<br/>
“**其他权限**”是指该角色可使用的系统后台的敏感数据控制权限。<br/>

## 操作步骤

**1.添加用户账号<br/>**
• 在“MaSe”导航栏中打开“**设置-基础设置-用户权限**”功能，进入“**用户权限**”页面。<br/>
![用户权限](/img/05backend/01backend.png)
• 在“**用户权限**”页面，点击右上角“**新增账号**”，<br/>
• 在“**添加账号**”页面，输入“**登录账号**”、“**登录密码**”、“**用户昵称**”，选择“**关联角色**”、“**绑定员工**”等内容后点击“**保存**”保存新账号。<br/>
![用户权限](/img/05backend/07backend.png)

**2.编辑用户账号权限<br/>**
• 在“**用户权限**”页面，点击要修改的账号后的“**权限**”按钮，打开“**编辑权限**”页面，<br/>
修改账号的权限内容，完成后点击“**保存**”完成设置。<br/>
![用户权限](/img/05backend/08backend.png)

**3.编辑用户账号信息<br/>**
• 在“**用户权限**”页面，点击要修改的账号后的“**编辑**”按钮，打开“**编辑账号**”页面，<br/>
修改账号内的内容，完成后点击“**保存**”完成设置。<br/>
![用户权限](/img/05backend/09backend.png)

**4.停用用户账号<br/>**
• 在“**用户权限**”页面，点击要修改的账号后的“**状态**”按钮，关闭账号状态，<br/>
![用户权限](/img/05backend/10backend.png)

---

## 温馨提示
1.用户角色：设定用户资料前，请先在角色管理中设定所有角色权限，以后设定用户权限时只需选择相对应的角色即可，如果该用户权限范围较多,还可以在用户权限中适当的调整；<br/>
2.员工在员工列表被**离职**之后，他的账号将**无法**再进行使用，**无法**登陆系统。<br/>
