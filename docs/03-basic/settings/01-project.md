# 项目资料管理

“项目资料”主要用于设定管理本店系统中服务项目、服务项目的价格及做项目给积分等。

## 操作步骤

**1.添加服务<br/>**
• 在“MaSe”导航栏中打开“**设置**-**基础设置**-**项目管理**”功能，进入“**项目列表**”页面。<br/>
![项目管理](/img/02settings/01settings.png)
• 首先点击右上角⚙️-**类别管理**，关闭自己不需要的类别，新建自己需要的类别。<br/>
![项目管理](/img/02settings/02settings.png)
• 点击右上角“**新增**”按钮打开“**添加新项目**”页面。填写“**项目名称**“，”**项目编号**“，“**项目分类**“，”**价格类型**“等相关内容，带有用红色”*️⃣“的为必填，其他为选填，完成后点击“**保存**”完成设置。<br/>
![项目管理](/img/02settings/03settings.png)


**2.编辑服务<br/>**
• 在“**项目列表**”页面，点击要修改的服务后的“**编辑**”按钮，修改里面的内容，完成后点击“**保存**”完成设置。<br/>
![项目管理](/img/02settings/04settings.png)

**3.快速设置服务<br/>**
• 在“**项目列表**”页面，点击右上角的“**快速设置**”按钮，选择“**设置范围**“，选择“**设置内容**“，输入要设置的值，输入”**确认**“点击“**确定**”按钮，可以批量修改服务的内容。<br/>
![项目管理](/img/02settings/08settings.png)

**4.导出服务<br/>**
• 在“**项目列表**”页面，点击右上角的“**导出为**”按钮，系统将自动导出所有服务为一个excel。<br/>
![项目管理](/img/02settings/09settings.png)
• 在系统主页右上角“**⏬**”按钮处，下载刚刚导出的报表<br/>
![项目管理](/img/02settings/10settings.png)

**5.停用服务<br/>**
• 在“**项目列表**”页面，点击要修改的服务后的“**状态**”按钮，关闭按钮后，该项目在“**收银**”，“**日程**“等业务页面都不会再显示，但不影响已购买权益继续消耗。<br/>
![项目管理](/img/02settings/05settings.png)

**6.关闭服务在线预约<br/>**
• 在“**项目列表**”页面，点击要修改的服务后的“**线上预约**”按钮，关闭按钮后，该项目在线预约时不会再出现，客人无法预约该项目。<br/>
![项目管理](/img/02settings/06settings.png)

**7.修改服务顺序<br/>**
• 在“**项目列表**”页面，点击要修改的服务前的“**排序**”的数字，**数字越小，排序越靠前**，该顺序影响**门店收银，和客人在线预约等**的服务展示顺序。<br/>
![项目管理](/img/02settings/07settings.png)
