# 今日提醒说明

今日提醒是方便您查看门店数据的窗口，您可以根据这些数据安排今日工作。<br/>

**计划到店**：代表项目设置中的”消费间隔“产生的客勤任务的数量。<br/>
**今日生日客户**：代表今日生日的会员人数。<br/>
**即将到期权益**：代表门店即将过期的权益卡的数量。 <br/>
**服务回访**：代表项目消费后生成的客勤任务的数量 <br/>
**客户流失**：代表系统生成的客户流失客勤任务数量（具体规则可在**设置**-**参数设置**-**客勤**中设置）。<br/>
**临期产品**：代表保质期到期的产品数量（按**产品管理**中设置的**保质期**天数进行计算。如开启**产品管理**中的**批次管理**，则根据有效期到期时间计算）<br/>
**临时客勤**：代表手动创建或系统触发的临时性客户跟进任务数量<br/>
**低库存产品**：代表当前库存等于或低于安全库存阈值的产品数量（根据**产品管理**中的“**库存警戒单位**”计算）<br/>

可点击对应字段后的 **“查看”** 按钮查看相应的明细
