# 每日数据
## 功能说明

首页上每日数据可以展示：<br/>

**现金类营业额**: 统计所有已选现金类支付方式的总额;<br/>
**营业额**: 统计所有已选支付方式的总额;<br/>
**预约数**: 当天预约数;<br/>
**卡付类营业额**: 统计所有已选权益类支付方式的总额;<br/>
**权益销售额**: 通过权益售卖所获得的金额;<br/>
**权益销售数**: 已售权益数;<br/>
**新增顾客数**: 顾客列表中新增的顾客数;<br/>
**到店数**: 到店顾客数;<br/>

---

## 功能入口

进入路径：

**系统后台** → **首页** → **每日数据**

可点击右上角齿轮按钮，选择需显示的数据，并设置显示排序，目前支持显示3类数据
