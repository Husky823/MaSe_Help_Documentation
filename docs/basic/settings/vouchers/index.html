代金券管理

“代金券”分为金额抵用和次数抵用，主要用于门店设置优惠促销活动，代金券可以免费赠送或者销售给顾客，顾客到店消费时使用代金券进行抵扣。

代金券共有两个分类：
1.储值代金券：客户购买或获赠后，账户中增加可抵扣的固定金额（如“50元代金券”），消费时按实际金额抵扣，支持设置有效期、单次抵扣上限等；
注：储值代金券只能使用一次，即使上次消费后有越剩余，也无法再次消费
2.次数代金券：客户购买或获赠后，获得指定服务项目的免费使用次数（如“1次美甲代金券”），每次消费对应项目时自动扣减1次，同样可设有效期及适用范围。

操作步骤​

1.添加代金券
 • 在“MaSe”导航栏中打开“设置-权益设置-代金券”功能，进入“代金券”页面。
• 首先点击右上角⚙️-标签管理，关闭自己不需要的标签，新建自己需要的标签。
• 点击右上角“新增”按钮打开“新增代金券”页面。填写“代金券编号“，”名称“，选择“代金券类型“，”销售价格“等相关内容，带有用红色”*️⃣“的为必填，其他为选填，完成后点击“保存”完成设置。

2.编辑代金券
 • 在“代金券”页面，点击要修改的代金券后的“查看”按钮。
• 再点击右上角“编辑“按钮，打开“编辑代金券”页面，修改里面的内容，完成后点击“保存”完成设置。

3.代金券同步已购客户
 • 修改代金券后，修改的内容并不会自动同步给已经售出的代金券，防止已售出的代金券资料混乱，影响门店账务。
• 如果需要同步修改内容到已售代金券，在“代金券”页面，点击要修改的代金券后的“查看”按钮。
• 再点击右上角“同步已购客户“按钮，“选择想要同步的内容”，点击右下角“保存”按钮，已修改的内容便会同步到已售代金券

4.快速设置代金券
 • 在“代金券”页面，点击右上角的“快速设置”按钮，选择“设置范围“，选择“设置内容“，输入要设置的值，输入”确认“点击“确定”按钮，可以批量修改代金券的内容。

5.导出代金券
 • 在“代金券”页面，点击右上角的“导出”按钮，系统将自动导出所有代金券为一个表格。
• 在系统主页右上角“⏬”按钮处，下载刚刚导出的报表

6.停用代金券
 • 在“代金券”页面，点击要修改的代金券后的“查看”按钮。
• 点击“使用状态”按钮，关闭使用状态，关闭按钮后，该代金券在“收银”等业务页面都不会再显示，但不影响已售代金券继续消耗。

温馨提示​

  1. 系统中已经使用的代金券不允许删除；
